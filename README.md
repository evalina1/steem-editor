# Steem Editor
Steem Editor is a smart editor for steem. Its purpose is to enhance the writing experience of writers in the steem community.

# How to Build

1. Pull repo

```
git pull https://github.com/ety001/steem-editor.git
```

2. Install composer

```
cd steem-editor
composer install
```

3. Run server

```
php artisan serve
```
